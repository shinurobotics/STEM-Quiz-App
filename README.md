# STEM Quiz App 🧠
A Python-based STEM quiz application developed as part of my programming learning journey. This project tests users on Science, Technology, Engineering, and Mathematics (STEM) concepts while demonstrating fundamental Python programming skills.

# About
The STEM Quiz App is a command-line application that presents users with multiple STEM-related questions, evaluates their answers, tracks their score, and provides feedback based on their performance.
This project was created to strengthen my understanding of Python fundamentals, user input handling, conditional statements, string manipulation, and score calculation.

# Features
- Interactive welcome screen
- Personalized user greeting
- Five STEM-related quiz questions
- Case-insensitive answer checking
- Real-time score tracking
- Instant feedback after each question
- Final percentage score calculation
- Performance evaluation system
- Completion message
- Randomized questions

# How It Works
1. User enters their name.
2. The quiz begins with five STEM questions.
3. Answers are checked against predefined solutions.
4. Correct answers increase the score.
5. The program displays the current score after every question.
6. A final percentage score is calculated.
7. Performance feedback is provided:
   - Excellent
   - Good
   - Try Again
