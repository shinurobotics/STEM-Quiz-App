# STEM Quiz App
A Python-based STEM quiz application developed as part of my programming learning journey.

## Features
- Interactive quiz experience
- Mathematics questions
- Physics questions
- Chemistry questions
- Biology questions
- Score tracking system
- Performance feedback

## Technologies Used
- Python

## How It Works
1. Enter your name
2. Answer STEM-related questions
3. Receive instant feedback
4. View your final score and performance rating

- Mathematics
